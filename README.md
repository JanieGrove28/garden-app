# 🌱 Garden App

## 📌 Overview
The Garden App is a simple Python program that provides gardening advice based on the month entered by the user. It helps users understand seasonal gardening tasks such as watering, pruning, planting, and frost protection.

This project was built to practice:
- GitHub repository management
- Issue tracking and planning
- Branching and pull request workflow
- Code refactoring and improvement

---

## 🌿 Features
- Provides seasonal gardening advice based on month input
- Clean and modular function-based structure
- Uses dictionary mapping for better data organisation
- Handles invalid inputs gracefully
- Improved readability and maintainability

---

## 🧠 How it works
The user enters a month, and the program:
1. Checks which season the month belongs to
2. Returns relevant gardening advice
3. Displays a message in the terminal

---

## 💻 Code Structure
- `get_season(month)` → Handles logic for returning seasonal advice  
- `main()` → Handles user input and program flow  
- Dictionary-based structure used for cleaner logic
